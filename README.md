# AstrBot AI绘画插件
基于竹和木的 https://github.com/3451996649/astrbot_plugin_ms_aiimg 插件优化而来

不懂编程全程使用ai生成代码。

## 🎨 功能特性
- **智能绘画触发**：在对话中自动识别绘画机会并生成图片
- **多模型支持**：支持魔搭社区、OpenAI格式等多种AI绘画模型
- **人物扮演集成**：结合人物形象生成更贴合的绘画提示词
- **独立模型配置**：判断、提示词生成、绘画三个功能可分别配置不同的大模型
- **对话历史感知**：基于完整对话上下文生成情境相关的画作

## 🚀 使用方法

### 1. 命令方式绘画
使用命令格式：`/aiimg <提示词>`
- 示例：`/aiimg 一只可爱的猫咪在花园里玩耍`

### 2. 智能绘画（自动触发）
在聊天过程中，机器人会根据对话内容自动识别绘画机会并生成图片
- 触发条件：包含绘画关键词或描述性内容
- 概率控制：可配置触发概率（默认30%）
- 间隔控制：最小绘画间隔时间（默认5分钟）

### 3. 人物扮演形象设置
使用命令格式：`/setcharacter <人物形象描述>`
- 设置后，智能绘画功能将结合人物形象和对话内容生成更贴合的绘画提示词
- 示例：`/setcharacter 一位穿着白色连衣裙的精灵少女，有着银色的长发和蓝色的眼睛，生活在森林中`

## ⚙️ 配置说明

### 绘画大模型配置
- **绘画大模型API地址**：文生图服务器地址（必填）
- **绘画大模型API密钥**：访问密钥（必填）
- **绘画大模型名称**：使用的AI模型名称
- **提供商类型**：模型提供商（modelscope/OpenAI）
- **分辨率**：生成图片的尺寸

### 智能判断配置
- **智能绘画触发概率**：聊天过程中触发绘画的概率（0-1之间的小数，默认0.3）
- **最小绘画间隔**：两次自动绘画之间的最小时间间隔（秒，默认300）
- **启用大模型智能判断**：是否启用AI大语言模型进行智能判断（默认关闭）

### 判断大模型配置（启用智能判断时使用）
- **判断是否绘画的大模型API地址**：用于智能判断的API地址（OpenAI格式）
- **判断是否绘画的大模型API密钥**：智能判断模型的访问密钥
- **判断是否绘画的大模型名称**：智能判断模型名称（默认gpt-3.5-turbo）

### 提示词生成大模型配置（启用智能判断时使用）
- **生成提示词的大模型API地址**：用于生成绘画提示词的API地址（OpenAI格式）
- **生成提示词的大模型API密钥**：提示词生成模型的访问密钥
- **生成提示词的大模型名称**：提示词生成模型名称（默认gpt-3.5-turbo）

### 人物扮演与对话历史
- **预先设定的人物扮演形象**：默认的人物形象描述
- **对话历史缓存大小**：保留的对话历史条数（默认10条）

## 🔧 技术特点

### 智能绘画判断机制
1. **关键词匹配**：识别"画"、"绘画"、"风景"、"人物"等23个关键词
2. **描述性内容识别**：检测形容词+名词的描述性结构
3. **AI智能判断**：使用独立的大模型进行更精准的判断

### 提示词生成优化
- **基于消息接收者视角**：从人物扮演角色的视角生成提示词
- **对话上下文感知**：结合最近的对话历史生成情境相关的提示词
- **人物形象契合**：确保生成的画作符合人物设定

### 多模型独立配置
- **判断模型**：专门用于判断是否应该绘画
- **提示词模型**：专门用于生成高质量的绘画提示词
- **绘画模型**：专门用于图像生成

```
用户：/aiimg 美丽的日落风景
机器人：[生成日落风景图片]
```

### 智能绘画触发
```

机器人：[自动生成公园花海的图片]
```

## ⚠️ 注意事项
- 魔搭API每天有一定的次数限制以及请求频率限制
- 启用智能判断功能需要配置相应的API地址和密钥
- 建议根据实际需求调整触发概率和间隔时间
- 人物扮演形象设置后，生成的图片会更贴合角色设定
